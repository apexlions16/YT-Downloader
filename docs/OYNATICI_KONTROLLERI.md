# Oynatıcı kontrolleri

## Android

- Oynat/duraklat
- Zaman çizgisi
- ±10 saniye
- Ses parçası
- Altyazı parçası veya kapalı
- 0.75×–2× hız
- Tam ekran

## Windows

- MPV görüntü penceresi
- Türkçe kontrol paneli
- Oynat/duraklat
- ±10 saniye ve başa dön
- Ses parçası
- Altyazı parçası veya kapalı
- Ses seviyesi
- 0.75×–2× hız
- Tam ekran
