# Bmobil ve BPC 0.3.0 geliştirme günlüğü

## Düzeltilen sorunlar

- Android'de şifreli dosya çözüldükten sonra oynatma dizininin oluşturulmaması düzeltildi.
- Android oynatıcıya gerçek ses, altyazı, hız, tam ekran ve sarma kontrolleri eklendi.
- Windows'ta indirme öncesinde internet üzerinden motor güncellemesinin zorunlu olması kaldırıldı.
- Windows JavaFX oynatıcı MPV tabanlı oynatıcıyla değiştirildi.

## Yeni medya özellikleri

- yt-dlp metadata çıktısından dublaj/alternatif ses formatları çıkarılır.
- Yayıncı altyazıları ve otomatik altyazılar ayrı gösterilir.
- Birden fazla ses ve altyazı seçildiğinde MKV kapsayıcısına gömülür.
- Android Media3 ve Windows MPV içinden ses/altyazı parçası değiştirilebilir.

## Dağıtımlar

- **Bmobil:** Android şifreli kütüphane
- **Bmobil Developer:** Android İndirilenler klasörüne açık medya
- **BPC:** Windows şifreli kütüphane
- **BPC Developer:** Windows seçilen diske açık medya

## Windows paketli motor

Kurulum dosyasında aşağıdaki bileşenler bulunur:

- yt-dlp Nightly
- FFmpeg
- FFprobe
- Deno
- MPV

Motor güncellemesi başarısız olsa bile paketli motor indirme ve oynatmayı sürdürür.
