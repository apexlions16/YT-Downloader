# Bmobil ve BPC 0.3.0 sürüm notları

- Windows indirme motoru paketli ve çevrimdışı başlangıçlı hâle getirildi.
- Android oynatma sorunu düzeltildi.
- Android Media3 ve Windows MPV oynatıcılarına ses/altyazı seçimi eklendi.
- İndirilecek alternatif ses/dublaj ve altyazı dilleri seçilebilir hâle getirildi.
- Çoklu parçalar MKV içine gömülüyor.
- Bmobil ve BPC şifreli sürümleri hazırlandı.
- Bmobil Developer ve BPC Developer açık dosya sürümleri eklendi.
- Dört ürün için otomatik APK, MSI, EXE ve GitHub Release hattı oluşturuldu.
